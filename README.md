# Car Scroll Animation

A smooth, scroll-based hero section animation inspired by premium 
automotive websites.

## Tech Used
- HTML
- CSS
- JavaScript
- GSAP + ScrollTrigger

## Features
- Full-screen hero section
- Intro text animations
- Scroll-linked vertical & horizontal motion
- Transform-based smooth animations
